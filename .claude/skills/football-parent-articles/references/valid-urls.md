## Rules

- Use only URLs from this list, or new article URLs created in the current batch.
- 3–5 contextual internal links inside the article body, placed behind natural anchor text.
- 3–4 links in the bottom "Related Articles" section, kept separate from body links.
- Never repeat the same internal link excessively in one article.
- Do not force a link where it interrupts readability — if it feels forced, cut it.
- Never guess or invent a slug. If unsure a URL exists, don't link it.
