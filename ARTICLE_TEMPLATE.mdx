## FAQ

Q: What should I include in an FAQ section?

A: Common questions parents ask about the topic. Usually 3-5 Q&As that address concerns not covered in the main sections.

Q: Are all these sections required?

A: No. Create sections based on your content. The only required sections are those listed in `frontmatter.sections` — keep those in sync with your actual `##` headings.

Q: How do I add this to the site?

A: Save this file as `content/[category]/[slug].mdx`, then create a corresponding `app/[category]/[slug]/page.tsx` that imports and renders it using `ArticleLayout`.
